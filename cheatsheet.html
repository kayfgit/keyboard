How it works: Hold keys together, release all to fire a chord. Left hand = consonant, right hand = vowel. One chord = one syllable. Space sends to AI for text conversion. C+M (both thumbs) = backspace. All 10 = enter.

Base Vowels 6 sounds

Right hand only (no M thumb)

Symbol Keys Sound English examples # keys

Lossy encoding: Each vowel covers multiple English sounds. "i" covers both /iː/ (see) and /ɪ/ (sit). "a" covers /æ/ (cat) and /ɑ/ (father). The AI uses context to determine the right word.

Modified Vowels M + base vowel

M thumb + same keys as base vowel = long/diphthong version

Symbol Keys Pattern AI interprets as # keys
ī M+; M + i iː (see, meet) or aɪ (eye, write) 2
ē M+K M + e eɪ (day, make, say) 2
ā M+J M + a ɑː (father, car, palm) 2
ō M+L M + o oʊ (go, home) or ɔː (thought, or) 2
ū M+L+; M + u uː (blue, food, true) 3
Learning tip: The modifier pattern is simple — press M (thumb) + the same keys as the base vowel. M+; = modified i, M+K = modified e, etc.
